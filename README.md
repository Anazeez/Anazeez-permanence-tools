# Project Permanence — Toolchain

> Deterministic tooling for the Project Permanence ecosystem.

## Purpose

Project Permanence Toolchain provides the generators, validators, templates, bootstrap utilities, and future compiler infrastructure used to transform governed specifications into deterministic artifacts.

The toolchain itself does **not** define constitutional truth.

Constitutional authority remains within the Pulse repository.

## Principles

- Specifications before implementation.
- Deterministic generation.
- Review before promotion.
- Explicit contracts.
- Verifiable outputs.
- Reproducible builds.
